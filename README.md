# Trial and Error

Learning to build with Claude Code, one chapter at a time.

This is a self-teaching curriculum that walks a complete beginner from zero through building a mini Eidrix-style app — tabs, chat UI, Supabase, AI, MCP — one small chapter at a time.

## How to use this repo

1. Clone this repo.
2. Open it in Claude Code.
3. Say: *"I'm new here, where do I start?"*
4. Claude Code will open `curriculum/chapter-0-first-session.md` and walk you through it.
5. Follow one chapter at a time. Check them off in `curriculum/PROGRESS.md` as you go.

No coding experience required. The curriculum assumes nothing.
